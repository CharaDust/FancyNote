#### Resultative Adverbial Clause

| Object | 释义         |
| ------ | ---------- |
|        | 如此……以至于\*  |
|        | 如此……以至于\** |
| —————  | ————————   |

| Object   | 释义            |
| -------- | ------------- |
|          | 因此以至于（正式）     |
|          | 达到……的程度（程度结果） |
|          | 到……的程度（范围）    |
|          | 到如此程度以至于（强调）  |
| ———————— | ————————————— |

Tips:
- \*` so ... that ` 中使用`形容词/副词`
- \*`such ... that` 中使用`名词`
