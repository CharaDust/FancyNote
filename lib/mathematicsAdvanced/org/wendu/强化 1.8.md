# 题型8 闭区间上连续函数的性质

**例23** 若 $f(x)$ 在 $[a,b]$ 上连续，$a < x_1 < x_2 < \cdots < x_n < b(n \geq 3)$。

证明：  
(1)存在一点 $\xi \in [a,b]$，使得  
$$f(\xi) = \frac{2[f(x_1) + 2f(x_2) + \cdots + n f(x_n)]}{n(n+1)}$$

(2)求  
$$\lim_{n \to \infty} \sqrt[n]{\frac{1}{n} \sum_{i=1}^{n} e^{f(x_i)}}$$
